## Cloud Career Relevance
These labs help build a foundational understanding of cloud and cloud security concepts, including:

- Basic cloud infrastructure configuration
- Network-level security controls in cloud environments
- Access control and exposure reduction
- Understanding common cloud misconfigurations
- Awareness of cloud security considerations relevant to SOC environments

This work supports preparation for entry-level roles such as:
- Cloud Support / Cloud Operations
- Junior Cloud or IT roles with cloud exposure
- SOC Analyst with basic cloud security awareness
